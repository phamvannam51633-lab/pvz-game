# 免疫学题库 UI 升级设计文档

## 概述

将 `mianyi-quiz.html` 从基础暗色主题升级为毛玻璃暗紫风格，优化首页布局、题目卡片、交互动效。保持现有答题逻辑、三端适配结构不变。

## 设计决策

### 视觉风格：毛玻璃暗紫 Glass Morphism

- 背景渐变 `#0f0c29 → #302b63 → #24243e`
- 主色调 `#7c4dff`（紫色），单选标签用紫色，多选用 `#e65100`（橙色）
- 卡片：`rgba(255,255,255,0.04-0.08)` 半透明 + `backdrop-filter: blur()` + 细边框
- 选中态：紫色填充背景 + 紫色边框，圆形字母标签变实心

### 题目卡片：A+C 融合

- 卡片带 `box-shadow` 悬浮阴影增加层次
- 选项使用圆形字母标签（直径 28px），选中时填充紫色
- 正确/错误反馈保留现有绿/红色方案
- 单选和多选标签通过颜色区分（紫 vs 橙）

### 首页：卡片式双栏

- 顶部图标 + 标题 + 副标题居中
- 两个模式卡片并排（手机端上下排列）：章节练习（主推，紫色边框高亮）、随机抽题
- Emoji 图标：📖 章节、🎲 随机
- 底部统计条：已刷 X/452 · 正确率 X%（数据存 localStorage）

## 保留不变

- 桌面端侧边栏章节列表、手机端下拉选择器
- 答题区 header（标题 + 进度）、#scroll 滚动区、#btn-row 按钮区
- `100dvh` 视口适配、三端媒体查询断点（≤768px 手机，≥769px 桌面）
- JavaScript 答题逻辑、进度计算、提交/重置
- `quiz-data.json` 数据结构和 `gen_html.js` 生成流程

## 新增微交互

- 选项 hover 时轻微上浮 `translateY(-1px)` + 边框变亮
- 进度条增加渐变发光效果
- 提交后正确率数字从 0 滚动到目标值
- 章节切换时题目区淡入过渡

## 文件变更范围

仅修改 `gen_html.js` 中的 CSS 和 HTML 结构，重新生成 `mianyi-quiz.html`。不改动 `quiz-data.json` 和解析脚本。

## 验收标准

- 桌面端、平板端、手机端三端视觉一致
- 所有答题功能正常（单选、多选、提交、重置、章节切换、随机模式）
- 首页统计条正确显示 localStorage 累计数据
- 选项点击、hover 动效流畅
